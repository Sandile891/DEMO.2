Robot Commands

move_forward(distance)
Move robot forward by specified distance (in cm)
move_backward(distance)
Move robot backward by specified distance (in cm)
turn_left(angle)
Turn robot left by specified angle (in degrees)
turn_right(angle)
Turn robot right by specified angle (in degrees)
stop()
Immediately stop all robot movements
set_speed(speed)
Set movement speed (1-10)
