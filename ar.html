Available Commands

  • forward(distance) Move robot forward by specified distance (units)
  • backward(distance) Move robot backward by specified distance
  • left(degrees) Turn robot left by specified degrees
  • right(degrees) Turn robot right by specified degrees
  • wait(seconds) Pause execution for specified seconds

About This Simulator

This interactive simulator allows you to program a virtual robot using Python. The robot will execute your commands on the virtual table surface, with realistic wheel rotation and movement physics.

All commands run in your browser - no server required! Start by modifying the sample code and click "Run Program" to see your robot in action.
