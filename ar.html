Example Programs

# Move in a square pattern
for i in range(4):
    move_forward(200)
    turn_right(90)
# Avoid obstacle
move_forward(150)
if read_distance() < 20:
    turn_left(90)
    move_forward(150)
# Follow light
while True:
    light = read_light()
    if light > 70:
        move_forward(100)
    else:
        turn_left(15)

Available Functions

move_forward(power)

Move robot forward at specified power (0-100)

move_backward(power)

Move robot backward at specified power (0-100)

turn_left(degrees)

Turn robot left by specified degrees

turn_right(degrees)

Turn robot right by specified degrees

stop()

Stop all robot movement

read_distance()

Get distance to obstacle in cm

read_light()

Get light sensor reading (0-100)

set_speed(speed)

Set default movement speed (0-100)

How to Use the Simulator

  1. Type Python commands in the terminal input field (e.g. move_forward(100))
  2. Click "Run" to execute your commands
  3. Watch the 3D robot model respond to your commands with physics-based movement
  4. Use the example programs as templates for your own creations
  5. Combine multiple commands to create complex behaviors
  6. Use loops and conditionals for more advanced programs

The simulator features realistic physics - wheels rotate proportionally to motor power, and turning is achieved by differential wheel speeds.
